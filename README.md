# nettools
Network tools implemented on C.

## Build
```bash
mkdir .build
cd .build
cmake ..
cmake --build .
```

### -or- (after CMake) For Linux with make:
```bash
make
```

### -or- (after CMake) For Windows:
```batch
MSBuild.exe nettools.sln
```

Executable examples can be found in "examples" and the corresponding nested directories.
